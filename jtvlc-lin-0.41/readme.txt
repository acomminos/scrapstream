Jtvlc is a cross-platform console application that allows you to broadcast on Justin.tv using VLC, a popular video player available for all three platforms (Windows, Mac, and Linux).

Please visit the Linux Broadcasting API page in our API Wiki for installation and usage instructions, including sample parameters for starting VLC and Jtvlc!

http://apiwiki.justin.tv/mediawiki/index.php/Linux_Broadcasting_API
